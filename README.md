Neste projeto, criei um sistema em Java para gerenciar as operações de controle de comandas de uma padaria. A interface foi desenvolvida utilizando JavaFX. O algoritmo é um exemplo de Programação Orientada a Objetos (POO), onde utilizei uma sequência de instâncias de objetos com herança, interface, etc. O algoritmo consiste em funções para controlar as comandas, permitindo criar, efetuar pagamento, adicionar itens, entre outras operações. Este projeto foi desenvolvido por mim, Cauã Pereira, como trabalho final da disciplina de Ferramentas Computacionais na faculdade.

In this project, I created a system in Java to manage the operations of controlling orders in a bakery. The interface was developed using JavaFX. The algorithm is an example of Object-Oriented Programming (OOP), where I used a sequence of object instances with inheritance, interface, etc. The algorithm consists of functions to control orders, allowing for creating, making payments, adding items, among other operations. This project was developed by me, Cauã Pereira, as the final assignment for the Computational Tools course in college.
